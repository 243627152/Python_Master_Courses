    - PythonAnywhere是一个基于Python语言的在线集成开发环境（IDE）和Web托管服务。它提供内嵌在浏览器中的相关接口（就是图形化界面），来访问服务中的基于服务的Python和Bash命令，以及带有高亮的代码编辑器。程序文件可以通过浏览器可以进行上传到服务和下载到本地。由服务托管的Web应用程序可以使用任何WSGI应用框架。
