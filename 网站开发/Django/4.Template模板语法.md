## 4.Template模板语法

- 传递变量到模板
    - 在views.py 中设置context字典
```python
context = {'title':'欢迎光临首页','name': 'xiaoMing','age':14,'class':'高一3班','scores':[73,84,68,93]}
#在index()
return render(request,template_name='helloworld/index.html',context=context)
```  
- 在模板文件helloworld/index.html中
```html
<title>{{ title }}</title>
    <ul>考试成绩：
        <li>姓名：{{ name }}</li>
        <li>年龄：{{ age }}</li>
        <li>班级：{{ class }}</li>
        <div>成绩
        <p>
            {% for scro in scores %}
                {{ scro }}  
            {% endfor %}
        </p>
        </div>
    </ul>
```
